# 🎯 Zgadywanka Liczb v1.0

Gra polegająca na odgadnięciu liczby od 1 do 100 wygenerowanej przez komputer.

## 🚀 Czego się tutaj nauczyłem?
* **Obsługa błędów:** Wykorzystanie `try...except`, aby program nie "padał" po wpisaniu litery.
* **Logika gry:** Zastosowanie pętli `while True` oraz instrukcji `break` do kończenia rozgrywki.
* **Biblioteki:** Importowanie i używanie `random.randint()`.
* **Git:** Zarządzanie zdalnymi repozytoriami (`git remote`) i rozwiązywanie problemów z wypychaniem kodu.

## 🛠️ Jak uruchomić?
Wpisz w terminalu:
`python zgadnij.py`
