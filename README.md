# Zgadywanka Liczb v1.0

Prosta gra konsolowa zgadywania liczb 1-100 Napisana w Pythonie, ucząca podstaw logiki i obsługi błędów.

## 🛠️ Zastosowane technologie
* **Python 3**
* **Moduł `random`** - do generowania losowych wyzwań.
* **Bloki `try...except`** - do obsługi błędnych danych wejściowych.

## 📖 Czego się nauczyłem?
1. **Pętle `while`**: Utrzymywanie programu w działaniu do momentu wygranej.
2. **Warunki `if/elif/else`**: Logiczne sterowanie podpowiedziami dla gracza.
3. **Git**: Zarządzanie wieloma repozytoriami i naprawianie błędów połączenia (remote).

## 🚀 Jak grać?
Wpisz w terminalu: `python zgadnij.py` i spróbuj odgadnąć liczbę w jak najmniejszej liczbie kroków!
